# 三篇论文对 GC-NKGraph-Atlas 的增益分析 + 落地方案

## Context

项目 `GC-NKGraph-Atlas`（BiB 投稿，现处 **Major Revision** 修回阶段）把已发表的
NK 免疫逃逸机制（Zheng et al., Nat Immunol 2023 的 serine→sphingomyelin(SM)→
membrane-topology→NK-cytotoxicity 轴）计算化复现，并扩展到胃癌（Arm B），肝/HCC 为
阳性对照（Arm A）。修回未决的**最致命科学缺口**（见 `submission_package/TDD_review_fixes.md`）：

- **T6 去循环**（★最易被攻击）：候选靶标排序自洽性、Table 4 与 Fig 3B 候选不一致。
- **H5 = DATA_UNAVAILABLE**：瘤内(IT) vs 瘤旁(PT) NK 的单细胞对照，锚定论文未存数据。
- **T14 H3 激活重叠**（已通过）：protrusion~cytotoxicity 需扣除通用 NK 激活签名。
- **T15 H2 效应量微小**：单细胞 r=+0.03，需以效应量+多重校正重述。
- 候选靶标图谱偏**表面受体 + 代谢酶**（SMPD3/SGMS1/HAVCR2…），缺 TF/胞内调控层。
- 缺**临床结局锚点**：NK 状态只对到标签，未对到生存/疗效。

用户目标：分析 `paper/` 下三篇论文对本项目的增益，产出**分析备忘录 + 落地方案**
（映射到修回任务 + 引用/改稿位置清单），**本轮不改稿件本体**。

## 增益评估（按增益从高到低）

### P3 — CLEC12B（Nat Immunol 2026）｜增益最高 ★★★★★

**核心内容**：CLEC12B（ITIM 型 C-lectin，胞外域与 NKG2D 同源）是**肿瘤浸润 NK 的抑制性
检查点**；配体为脂蛋白脂肪酶(LPL)；纳米抗体阻断 CLEC12B–LPL 轴可复活 NK、与 PD-1 阻断
协同；189 例 HCC scRNA + 789 例 bulk + TCGA-HCC。

**对本项目的增益**：
1. **直接填 H5 缺口**：该文有 **IT vs PT NK scRNA**，显示瘤内 NK 检查点/耗竭程序更高、
   细胞毒通路下调——正是我们 H5 标 `DATA_UNAVAILABLE` 的对照。可作**外部佐证**，把 H5
   从"数据缺失"改述为"外部独立队列支持该方向"。
2. **独立验证 HAVCR2/TIM-3 候选**：该文在 **TCGA-HCC** 中显示 CLEC12B/CD56 与
   **HAVCR2/TIM-3、TIGIT、PDCD1、LAG-3 正相关**——与我们候选逻辑同队列、同共表达口径，
   为 Top-8 里的 HAVCR2 提供外部支撑（缓解 T11 弱值措辞压力）。
3. **新候选 + 概念验证**：CLEC12B 本身是一个 2026 年新验证的 NK 检查点靶点（可药、纳米抗体、
   PD-1 协同），可作为"机制卡片→靶点"引擎产出的**同类新鲜范例**写入讨论（注意：它属表面
   受体、不在 SST/鞘磷脂轴内，定位为"NK-dysfunction 靶点空间的并行证据"，不可混入 SST 轴）。
4. **可信度**：同刊系(Nat Immunol)、同病种(HCC=我们 Arm A 肝对照)、同数据(TCGA-HCC)、最新
   （2026）——审稿人视角下极具说服力。

**局限**：机制是受体-配体（CLEC12B-LPL），与我们的鞘磷脂/膜拓扑轴**不同机制**，只能并列
不能等同；勿声称它验证了 SST 轴本身。

### P2 — CREM（Nature 2025）｜增益中高 ★★★★

**核心内容**：转录因子 CREM 是 CAR/IL-15 NK 的**负调控检查点**；PKA-CREB-CREM 轴，经表观
重编程抑制 NK；CREM 敲除增强 CAR-NK 效应与抗耗竭。

**对本项目的增益**：
1. **补齐候选图谱的 TF/胞内层**：我们候选偏表面受体+代谢酶，缺**转录因子/胞内负调控**
   这一类。CREM/PKA-CREB 是当前排序漏掉的类别，可据此在讨论中提"候选图谱应扩展到调控 TF
   层"，或作为 limitation/future-work。
2. **强化引言动机**：该文原话"NK 功能障碍机制研究有限、多聚焦表面受体、TF 层被忽视"几乎
   与我们引言框架一致——优质引言引文。
3. **支撑 T14 决策**：该文观察 NK 同时呈现**激活+功能障碍双签名**——正是我们 T14 处理
   protrusion~cytotoxicity 激活重叠的现象学依据，佐证"必须扣激活签名"的方法选择。
4. **靶点范式先例**：CREM KO 增强 NK = "找 NK 内在负调控子→靶向"的成功先例，支撑本文论点。

**局限**：体系是 CAR-NK（工程化），非原位肿瘤 NK；机制与 SST 轴无直接交集，属框架/方法层增益。

### P1 — 单细胞 CAR-T 图谱（Nature 2024）｜增益中｜方法/框架为主 ★★★

**核心内容**：695,819 个输注前 CAR-T 的单细胞多组学图谱；**type-2 功能**与 8.4 年 B 细胞
缺失（持久缓解）关联；配体-受体分析显示 type-2 亚群调控功能障碍亚群维持稳态；IL-4 引导可
改善耗竭；血清 type-2 细胞因子在长期应答者更高。

**对本项目的增益**：
1. **"Atlas→功能态→临床结局"金标准范式**：我们名为 *Atlas* 但只把 NK 态对到标签、未对到
   结局。该文是把单细胞功能态锚定到长期疗效的模板，可用于引言/讨论定位，并作为
   "缺结局锚点"这一 limitation 的对照与 future-work 方向。
2. **支撑图/配体-受体建模选择**：该文用 L-R 相互作用解释"一个亚群调控功能障碍亚群"——与我们
   异质图的 cell-cell / `metabolic_crosstalk` 边同构，可与 T7 消融结论互为独立支持。
3. **方法学参照**：大规模单细胞多组学图谱 + L-R 分析的规范做法。

**局限**：对象是 CAR-T 非 NK，机制（TH2/IL-4）与本项目无直接机制交集；增益主要在框架、方法、
叙事定位，科学证据增量弱于 P2/P3。

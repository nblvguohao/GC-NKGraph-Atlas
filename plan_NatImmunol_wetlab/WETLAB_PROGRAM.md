## 1. 背景与科学缺口(给湿实验读者)

**已被证实的机制(Zheng et al., *Nat Immunol* 2023,肝癌):** 肿瘤失调丝氨酸代谢 → 降低 NK 细胞
鞘磷脂(SM)合成底物 → NK 膜 SM 耗竭 → 膜突起/微绒毛塌陷 → 无法形成裂解性免疫突触 → 杀伤丧失。
该表型可被抑制 SM 分解(ASM/SMPD1 或 NSMASE/SMPD2-4)挽救,并与 Tim3(HAVCR2)阻断协同。
该文用超分辨 SEM + 单免疫细胞质谱在小队列证明,**方法不可规模化,未沉积转录组数据**,
且扩展癌种仅列肺/结肠/卵巢——**未含胃癌**。

**计算工作(GC-NKGraph-Atlas)发现并留给实验的关键点:**
1. 轴的**效应臂**(突起机器 ↔ 细胞毒性)在公共转录组可稳健恢复,并**泛化到胃癌**(Zheng 未列)。
2. **一个机制悖论 —— NI 级核心问题:** 瘤内 NK 的**突起机器基因转录反而升高**,而**杀伤功能降低**、
   物理拓扑塌陷。即**转录层与物理拓扑层解偶联**:转录不能代理物理表型。
3. 由此假说:**是转录后/翻译后的 SM 耗竭在膜层面使机器失能,而非下调其转录。**

**本计划要回答的 NI 级问题:**
> 在突起机器转录完好甚至升高时,是什么在膜/翻译后层面塌陷了 NK 突起并废除杀伤?
> 该机制是否从肝癌扩展到胃癌?能否据此给出可转化的联合治疗?

## 2. 中心假说与总体设计

**中心假说:** 肿瘤丝氨酸程序 →(代谢物层)NK SM 耗竭 →(翻译后)突起机器膜募集/组装失能 →
突起塌陷 → 杀伤丧失;该轴在**胃癌**中成立,并可被"SM 恢复 + Tim3 阻断"协同逆转。

**四个 Aim(人+鼠、体外+体内、损失+获得功能、含挽救实验):**
- Aim 1 在胃癌 NK 确立轴 + 直接证明转录–拓扑解偶联
- Aim 2 因果链(含框架点名酶 SGMS2/SMPD3/SMPD1、PHGDH/PSAT1/PSPH)
- Aim 3 体内治疗组合(SM 恢复 + Tim3 阻断,胃癌模型)
- Aim 4 临床验证与分层生物标志物

---

## 3. Aim 1 — 在胃癌 NK 确立轴,并直接证明"转录–拓扑解偶联"

**检验的计算预测:** 效应臂泛化胃癌(Arm B);瘤内 NK 细胞毒↓;转录不代理拓扑(机器 mRNA↑ 而功能↓)。

**样本:** 胃癌患者配对新鲜样本(肿瘤 / 癌旁正常 / 外周血),流式分选 NK(CD3⁻CD56⁺;分 CD56^bright/dim)。
n≥20(组学),n≥6(成像);预注册功效分析与样本量。

**在同一批 NK 上并行四类测量(关键:同细胞群同时测转录与拓扑):**
| 读出 | 方法 | 目的 |
|------|------|------|
| 物理拓扑 | 超分辨 SEM + SIM / 晶格光片,量化微绒毛/突起密度与长度 | 复刻 Zheng 金标准 |
| 突起机器转录 | scRNA-seq 或 panel qPCR(ERM、Arp2/3、WASP/WAVE、Rho GTPase、formin、BAR) | 测转录方向 |
| 膜 SM 含量 | 分选 NK 脂质组学(LC-MS/MS)或单免疫细胞质谱 | 测代谢物层 |
| 功能 | 对胃癌系(AGS、MKN45、NCI-N87、MKN28)共培养杀伤 + 免疫突触(perforin/F-actin 极化) | 测功能终点 |

**预期与判读:** 瘤内胃癌 NK 呈"转录不降甚至↑ + 拓扑塌陷 + SM 耗竭 + 杀伤↓" → **直接实证解偶联悖论**,
确立胃癌轴扩展。
**Go/No-Go:** 须在胃癌重现肝癌方向的拓扑塌陷;否则轴扩展不成立,Aim 2–4 暂停或改向。

---

## 4. Aim 2 — 因果链(双向扰动 + 挽救,含框架点名酶)

**检验的计算预测:** metabolic_crosstalk 因果假设;解释为何转录层代谢耦合"可忽略"(实为代谢物/翻译后层效应)。

**4a 肿瘤侧扰动(丝氨酸程序):**
- PHGDH 抑制(NCT-503 或遗传 KO/KD)、PSAT1/PSPH KD 于胃癌系;丝氨酸/甘氨酸剥夺培养基。
- ¹³C-丝氨酸代谢流示踪 → 确认对 NK 鞘脂合成底物供给的影响(实证"转录≠通量")。
- 下游读出:共培养 NK 的 SM(脂质组学)、拓扑(成像)、杀伤。

**4b NK 侧因果(框架点名酶,损失 + 获得功能):**
- CRISPR KO **SGMS2**(SM 合成)、**SMPD3 / SMPD1**(SM 分解)于原代 NK 或 NK-92;测拓扑 + 杀伤。
- 获得功能:SGMS2 过表达 / 外源 SM 补充 → 检验能否**挽救**拓扑与杀伤。
- 药理挽救:SMase 抑制 **ARC39**(ASM/SMPD1)、**GW4869**(NSMASE/SMPD2-4)。

**4c 机制深度(NI 级关键):** 在突起机器**蛋白**层面(WB 总蛋白 + 膜分选 + 活细胞成像定位),
证明 SM 耗竭使机器在**膜募集/组装**层面失能,而 mRNA/总蛋白不降 → 正面解答 Aim 1 的悖论。

**判读:** 建立完整因果链并解释计算层"代谢耦合可忽略"的原因。
**Go/No-Go:** SMase 抑制或 SM 补充须能挽救拓扑 + 杀伤;否则代谢因果假设不成立。

---

## 5. Aim 3 — 体内治疗组合(SM 恢复 + Tim3 阻断,胃癌)

**检验的计算预测:** 治疗假说(SM 恢复 + Tim3 阻断协同)从肝癌扩展到胃癌;分层读出。

**模型(≥2 类,人鼠互证):**
1. 免疫健全同基因胃癌模型(如 MFC / YTN16 于对应品系)——保留内源 NK。
2. 人源化模型(NSG + 人 NK 过继 + 胃癌 PDX 或细胞系)——检验人 NK。
3.(可选)NK 过继治疗:工程化(SGMS2↑ 或 SMPD KO)NK 的治疗增益。

**臂:** 对照 / SMase 抑制(恢复 NK SM)/ 抗 Tim3 / **组合**。
**读出:** 肿瘤生长与生存;瘤内 NK 浸润与功能(流式、scRNA);体内/离体 NK 拓扑成像;
**NK 依赖性对照**(anti-asialo-GM1 或 anti-NK1.1 耗竭,证明疗效依赖 NK)。

**判读:** 组合显著优于单药且依赖 NK → 把肝癌结论扩展到胃癌,给出可转化联合策略(NI 级转化落点)。
**Go/No-Go:** 组合须显协同且 NK 依赖;NK 耗竭后疗效消失为关键阳性对照。

---

## 6. Aim 4 — 临床验证与分层生物标志物(锚定结局)

**检验的计算预测:** 分层读出(NK SM-catabolism score + HAVCR2);弥补计算工作"无临床结局锚定"的局限。

- **回顾性:** 带随访的胃癌队列,检验分层读出与生存/免疫治疗应答的关联(独立于计算建模队列)。
- **原位空间验证:** 胃癌 TMA 多重免疫荧光 / 空间组学(CODEX / Xenium),原位共定位
  NK 拓扑标志 + SM 酶(SGMS2/SMPD3)+ Tim3,证明轴在组织空间中的共变。
- **前瞻(最强):** 配对 bulk RNA-seq + scRNA-seq + 功能 NK 检测 + 拓扑成像的专用胃癌队列。

**判读:** 分层读出若预测结局/应答 → 从 in-silico 假设升级为候选临床生物标志物。

## 7. 贯穿始终的 NI 级严谨性要求

- 生物学重复 + 技术重复;成像与打分**盲法**;预注册功效分析与样本量。
- 多个胃癌系 + 原代患者样本;**损失功能 + 获得功能**双向;**挽救实验**确立因果。
- 体内 **NK 依赖性**(耗竭对照);**人 + 鼠**互证;拓扑用 **≥2 种正交成像模态**。
- 每个实验显式标注其验证/证伪的计算预测(见第 8 节)。

## 10. 最小可发表单元与交付物

- **最小可发表单元(资源受限时):** Aim 1 + Aim 2 + Aim 3 的至少一个体内模型,
  即足以支撑一篇聚焦"胃癌 NK 丝氨酸–SM–拓扑轴 + 转录–拓扑解偶联 + 可转化联合"的 NI 级投稿;Aim 4 增强临床说服力。
- **候选靶点清单(来自计算工作,供 Aim 2 选择):** PHGDH、SGMS2、PSAT1、PSPH、SMPD3、SMPD1(核心);
  ERBB2、FGFR2、MET、MICA(胃癌相关/应力配体,次要)。
- **关键抗体/试剂候选(执行方须按本地确认):** 分选 CD3/CD56;SMase 抑制 ARC39、GW4869;
  PHGDH 抑制 NCT-503;抗 Tim3(HAVCR2);NK 耗竭 anti-asialo-GM1 / anti-NK1.1。
